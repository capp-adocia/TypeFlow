# TypeFlow
`INTRUCTION OF CHINESE OR ENGLISH`

### 这是什么？
TypeFlow 是一个酷炫的打字速度可视化工具！它就像你键盘旁的"心跳监测器"，用三个会变色的霓虹灯圆点告诉你打字有多快。

### 它能做什么？
- **实时颜色变化**：打字的字慢，圆点是灰色；正常速度变绿色；打得飞快就变红色
- **三个小圆点**：上面的圆点先变色，下面的跟上，像波浪一样流动
- **自动隐藏**：平时很窄，鼠标放上去才变宽显示提示
- **始终置顶**：不会挡着你打字，按ESC就能退出

### 使用场景
- **程序员写代码**：看看自己敲代码的速度节奏
- **装点桌面**：酷炫的RGB灯效装饰

### 安装
```bash
pip install PyQt5 pynput
python main.py
```

试试看，让你的打字变成一场灯光秀！

# TypeFlow  

### What is TypeFlow?
TypeFlow is a cool typing speed visualizer! Think of it as a "heartbeat monitor" for your keyboard—three neon dots that change color based on how fast you type.

### What does it do?
- **Real-time color changes**: Grey when idle, green at normal speed, red when typing fast
- **Three glowing dots**: Top dot changes first, creating a wave-like flow effect
- **Auto-hide**: Stays narrow until you hover to see details
- **Always on top**: Never gets in your way, press ESC to exit

### Perfect for
- **Coders**: Visualize your coding rhythm
- **Desktop bling**: Adds cool RGB-like lighting to your setup

### Start
```bash
pip install PyQt5 pynput
python main.py
```

Turn your typing into a light show!
